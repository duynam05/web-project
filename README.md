#  Đề tài: Xây dựng Website Bán Sách

---

##  Tên đề tài

**Xây dựng website bán sách**

---

##  Mô tả sơ qua dự án

Đây là dự án xây dựng website bán sách online, hỗ trợ người dùng tìm kiếm, xem thông tin và mua sách trực tuyến. Hệ thống hướng tới việc cung cấp các chức năng quản lý tài khoản, sản phẩm, giỏ hàng, đơn hàng, thanh toán.

Dự án được phát triển nhằm phục vụ mục đích học tập, đồng thời mô phỏng hệ thống thương mại điện tử cơ bản. Một số chức năng có thể được mở rộng thêm theo định hướng thực tế.

---

##  Các chức năng nổi bật trong dự án

| Nhóm chức năng       | Mô tả ngắn                                     |
| -------------------- | ---------------------------------------------- |
|  Xác thực người dùng | Đăng ký, đăng nhập, đăng xuất, mã hóa mật khẩu |
|  Quản lý sách        | Hiển thị danh sách sách, xem chi tiết          |
|  Tìm kiếm            | Tìm kiếm sách theo tên, thể loại               |
|  Giỏ hàng            | Thêm, xóa, cập nhật số lượng sản phẩm          |
|  Đặt hàng            | Nhập thông tin, tạo đơn hàng                   |
|  Thanh toán          | Thanh toán trực tuyến                          |
|  Người dùng          | Quản lý thông tin cá nhân                      |

## 🛠️ Công nghệ sử dụng  

###  Frontend
- HTML5, CSS3  
- JavaScript  
- ReactJS (xây dựng giao diện người dùng)

###  Backend
- Java  
- Spring Boot (xây dựng RESTful API)  
- Spring MVC  
- Spring Data JPA / Hibernate 

### 🔹 Database
- MySQL

### 🔹 Xác thực & bảo mật
- Spring Security  
- JWT (JSON Web Token)  
- BCrypt (mã hóa mật khẩu)

### 🔹 DevOps & Triển khai
- Docker   
- Docker Compose 
- Apache Tomcat

### 🔹 Công cụ phát triển
- Git & GitHub (quản lý mã nguồn)  
- Postman (test API)  
- Visual Studio Code (code editor)
- IntelliJ IDEA / Eclipse  


### 🔹 Khác
- RESTful API  
- MVC Architecture  
- Responsive Design
