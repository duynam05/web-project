# Đề tài: XÂY DỰNG WEBSITE BÁN SÁCH

## Giới thiệu

**Tên đề tài:** Xây dựng website bán sách

Đây là dự án xây dựng website bán sách online, hỗ trợ người dùng tìm kiếm, xem thông tin và mua sách trực tuyến. Hệ thống hướng tới việc cung cấp các chức năng quản lý tài khoản, sản phẩm, giỏ hàng, đơn hàng và thanh toán.

Dự án được phát triển nhằm phục vụ mục đích học tập, đồng thời mô phỏng một hệ thống thương mại điện tử cơ bản. Trong tương lai, một số chức năng có thể tiếp tục được mở rộng theo định hướng thực tế.
