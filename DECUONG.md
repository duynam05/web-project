# ĐỀ CƯƠNG ĐỀ TÀI

## 1. Tên đề tài
Xây dựng Website Bán sách

## 2. Mô tả đề tài
Xây dựng một hệ thống website bán sách cho phép người dùng xem sản phẩm, đặt hàng và thanh toán trực tuyến. 
Hệ thống hỗ trợ quản trị viên quản lý sản phẩm, đơn hàng và người dùng.

## 3. Đối tượng sử dụng
- Quản trị viên (Admin)
- Người dùng (User/Customer)

## 4. Chức năng hệ thống

### 4.1. Admin
- Quản lý người dùng
  - Thêm / sửa / xóa user
- Quản lý sản phẩm
  - Thêm / sửa / xóa sản phẩm
- Quản lý đơn hàng
- Thống kê / báo cáo
  - Doanh thu
  - Số lượng đơn hàng

### 4.2. User
- Trang chủ
- Xem danh sách sản phẩm
- Tìm kiếm sản phẩm
- Xem chi tiết sản phẩm
- Giỏ hàng
- Đặt hàng
- Thanh toán online
- Quản lý đơn hàng
- Đăng ký / đăng nhập
- Cập nhật thông tin cá nhân

## 5. Công nghệ dự kiến sử dụng
- Frontend: HTML, CSS, JavaScript (ReactJS nếu có)
- Backend:  Java
- Database: MySQL
